Who am I?

I'm a Graphic Designer and Music Composer.

Saya seseorang yang memiliki ketertarikan dengan desain grafis, sinematografi, animasi, composing musik, serta pembuatan dan pengembangan sistem informasi. Menjadi seorang yang berkompeten pada ketertarikan yang saya sebutkan di atas merupakan sebuah impian.

Saya senang mempelajari hal baru secara otodidak. Untuk mempelajari hal baru, saya lebih suka memanfaatkan media internet. Saat ini, saya lebih cenderung menguasai desain grafis dan pemrograman web dengan tidak mengesampingkan ketertarikan saya yang lain untuk terus mempelajarinya.

Graphic Designer

Skill utama saya adalah desain grafis. Selepas lulus dari SMK, sembari menunggu pengumuman SBMPTN saya bekerja di tempat reklame/advertising. Pengalaman saya di tempat tersebut yaitu merancang dan membuat logo serta mendesain output yang bagus untuk hasil cetak. Keahlian saat ini : CorelDraw, Adobe Family.

Music Composer

Saya mempunyai sedikit skill mengaransemen musik. Skill ini saya dapat secara otodidak sewaktu SMK. Saya sama sekali tidak ada basic bermain musik. Hanya bermodalkan niat, saya mencoba memahami step by step merancang dan mendesain sebuah karya musik. Untuk saat ini, fokus saya pada genre Instrumental/Klasik Orkestra/EDM/Mix both of them. Dengar beberapa musik saya di Spotify

Web Developer

Namun ketika mengikuti LKS (Lomba Kompetensi Siswa) pada divisi Web Design & Development sewaktu SMK, tidak disangka saya meraih juara II di tingkat kota Surakarta.
